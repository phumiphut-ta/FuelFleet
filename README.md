# FuelFleet™ - ระบบบริหารจัดการรถส่วนกลางและควบคุมโควต้าน้ำมันเชื้อเพลิง

ระบบเว็บแอปพลิเคชันบริหารจัดการยานพาหนะหลวง (รถยนต์ส่วนกลาง) และควบคุมยอดโควต้าน้ำมันเชื้อเพลิงรายคันแบบเรียลไทม์ ออกแบบมาในสไตล์ Luxury Dark Mode + Glassmorphism รองรับการตรวจสอบข้อมูลและการจองใช้งานผ่าน LINE LIFF บนอุปกรณ์พกพา และมีระบบควบคุมหลังบ้าน (Admin Console) เต็มรูปแบบ

---

## 🌟 คุณสมบัติเด่นของระบบ (Core Features)

* **ปฏิทินจองรถยนต์ส่วนกลาง (Central Booking Calendar)**: ปฏิทินแสดงรอบการจองในรูปแบบรายวัน/รายเดือน สีสันแยกแยะตามยอดจองของรถแต่ละคัน (Color-coded Events)
* **หน้าจอสืบค้นข้อมูลสำหรับผู้ใช้งานทั่วไป (LINE LIFF)**: ออกแบบสไตล์โมบายล์เฟิร์ส (Mobile-first Layout) สำหรับเปิดดูบน LINE LIFF แสดงทะเบียนรถ, โควต้าเต็ม, ปริมาณเติมสะสม และปริมาณคงเหลือของรถยนต์ทุกคันแบบลดลงตามสัดส่วนการใช้น้ำมัน (Draining bar 100% -> 0%) และเตือนสถานะใช้เกินโควต้า
* **ศูนย์ประมวลผลสถิติและรายงาน (PDF Reports Center)**:
  1. รายงานการใช้น้ำมันรายเดือน
  2. รายงานสถิติการใช้น้ำมันรถยนต์ส่วนกลางรายปีงบประมาณ (แยกแยะสะสมเป็นรายเดือน)
  3. รายงานจังหวัดจุดหมายปลายทางยอดนิยมประจำปีงบประมาณ
  4. รายงานประวัติคำสั่งระงับการใช้รถชั่วคราว
  5. รายงานประวัติความปลอดภัยหลังบ้าน (Audit Logs)
  6. รายงานใบเสร็จค่าน้ำมันประจำเดือนจำแนกรายคันพร้อมพิมพ์ใบสำคัญคู่จ่าย (PDF/Voucher Certification)
  7. รายงานการใช้น้ำมันรายปีงบประมาณแยกแยะรถยนต์หลวงแต่ละคัน
  8. รายงานสถิติผู้จองใช้งานรถยนต์ส่วนกลางเรียงลำดับรอบใช้งานจากมากไปน้อย
* **แผงควบคุมหลังบ้านแอดมิน (Admin Dashboard Console)**:
  * จัดการคลังจองรถ แก้ไข/ปรับเปลี่ยนเส้นทางและจังหวัดปลายทาง (รองรับ 77 จังหวัด)
  * ยกเลิกการจองรถยนต์หลวงโดยผู้ดูแลระบบโดยตรง (Passwordless Cancellation)
  * ตั้งค่าโควต้าน้ำมันรายเดือนรายรถ และระบบระงับใช้รถชั่วคราว
  * การอนุมัติตรวจสอบใบเสร็จค่าน้ำมันและภาพหลักฐานแนบ (รองรับไฟล์ PDF และรูปภาพ)
  * แผงเปลี่ยนรหัสผ่านแอดมิน (Bcrypt Hashing) และหน้าตรวจสอบสถิติการเดินทางยอดนิยม (Travel Analytics Dashboard) ร่วมกับ Chart.js
* **ระบบจัดการความเสถียร (Robust Core Framework)**: รองรับการนำทางภายใต้ Sub-directory (IIS Sub-Application) และระบบจัดเก็บ Log การทำงานอย่างปลอดภัย

## 🛠️ เทคโนโลยีที่ใช้พัฒนา (Tech Stack)

* **Backend**: PHP 8.3 หรือสูงกว่า (PSR-4 Autoloading)
* **Frontend**: HTML5, Vanilla CSS3 (Custom Glassmorphism Variables), Tailwind CSS (Play CDN), Alpine.js
* **Libraries**:
  * **mPDF**: สำหรับการสร้างรายงานเอกสาร PDF ภาษาไทยที่เป็นทางการ (Garuda Font)
  * **PhpSpreadsheet**: สำหรับการส่งออกข้อมูลการจอง/ใบเสร็จเป็น Excel (Lightweight BOM CSV)
  * **Chart.js**: สำหรับสร้างกราฟสรุปพฤติกรรมการเดินทางของผู้จองสูงสุด
  * **FullCalendar**: ปฏิทินแสดงอีเวนต์จองรถยนต์
  * **Leaflet.js**: สำหรับวาดและแสดงแผนที่ความร้อนความถี่การเดินทางรายจังหวัดปลายทางในหน้า Heat maps
* **Testing**: PHPUnit 10+ สำหรับทำ Unit Testing ระบบตรวจสอบโควต้าและเงื่อนไขจองซ้อน

## 🚀 ขั้นตอนการติดตั้งและใช้งาน (Installation & Setup)

### 1. โคลนคลังโค้ดและติดตั้งโปรแกรมประกอบ (Dependencies)
```bash
# Clone คลังซอร์สโค้ด
git clone <repository-url>
cd FuelFleet

# ติดตั้ง Composer packages
composer install
```

### 2. ตั้งค่าไฟล์ตัวแปรสภาพแวดล้อม (Environment Config)
คัดลอกไฟล์ต้นแบบ `.env.example` ไปตั้งชื่อเป็น `.env` และเปลี่ยนรายละเอียดการเชื่อมต่อ MySQL ตามเครื่องเซิร์ฟเวอร์ของคุณ:
```bash
cp .env.example .env
```
เปิดไฟล์ `.env` และกรอกรหัสผ่าน เช่น:
```env
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=fuel_fleet
DB_USERNAME=root
DB_PASSWORD=yourpassword
```

### 3. นำเข้าโครงสร้างฐานข้อมูล (Database Import)
นำเข้าสคริปต์สกีมาตั้งต้นที่มีการเตรียมและเพิ่มบัญชีผู้ใช้จำลองไว้เรียบร้อยแล้ว:
```bash
mysql -u root -p fuel_fleet < database/schema.sql
```
*(บัญชีสำหรับเข้าใช้งานแผงแอดมินตัวอย่าง: ชื่อผู้ใช้ `admin` / รหัสผ่าน `admin123`)*

### 4. รันระบบจำลองเครื่องโลคัล (Local Development Server)
สามารถรัน PHP Built-in Server ไปยังโฟลเดอร์ `/public/` เพื่อเปิดเข้าชมโปรเจกต์:
```bash
php -S localhost:8000 -t public
```

### 5. การเปิดใช้งานและลิงก์นำทาง
* **หน้าปฏิทินจองรถยนต์สาธารณะ**: `http://localhost:8000/`
* **หน้าสืบค้นโควต้า LINE LIFF บนอุปกรณ์พกพา**: `http://localhost:8000/liff/quotas`
* **หน้าแผงควบคุมหลังบ้านแอดมิน**: `http://localhost:8000/admin/login`
