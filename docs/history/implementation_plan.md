# แผนงานแสดงแผนที่ความถี่การเดินทางปลายทางในประเทศไทย (Thailand Travel Choropleth Heatmap Plan)

เราจะทำการพัฒนาและแสดงแผนที่ประเทศไทยแบบโต้ตอบได้ (Interactive Choropleth Map) ในหน้า Heat maps เพื่อแสดงความถี่ของการจองรถเดินทางไปยังจังหวัดปลายทางต่างๆ โดยใช้สีสันเข้ม/อ่อนตามจำนวนทริปการเดินทาง พร้อมทั้งปรับโครงสร้างเลย์เอาต์เพื่อการแสดงผลที่หรูหราและตอบสนองได้ดีเยี่ยม (Premium Glassmorphism & Responsive Design)

---

## ส่วนที่ผู้ใช้ต้องตรวจสอบและพิจารณาอนุมัติ (User Review Required)

> [!IMPORTANT]
> - **การเลือกใช้แผนที่แบบไม่ต้องเชื่อมต่อเซิร์ฟเวอร์กระเบื้อง (Offline Vector Map Rendering)**:
>   - เราจะใช้ไลบรารี Leaflet.js ร่วมกับข้อมูลเวกเตอร์เขตจังหวัดของไทยแบบย่อส่วนประสิทธิภาพสูงจากไฟล์โลคัล `thailand_apisit.json` (ขนาด 1.1MB, มี 77 จังหวัดครบถ้วนรวมบึงกาฬ) เพื่อให้สามารถวาดลายเส้นแผนที่ขึ้นมาในคอนเทนเนอร์แบบโปร่งแสงประสานกับธีมกลาสมอร์ฟิซึมได้ทันทีและโหลดได้อย่างรวดเร็ว โดยไม่จำเป็นต้องโหลดรูปแผนที่ถนนจริงจากเซิร์ฟเวอร์ภายนอกที่อาจขัดแย้งกับธีมมืดและลดทอนประสิทธิภาพ
> - **โทนสีการเน้นเขตจังหวัด (Color Gradient & Highlights)**:
>   - เพื่อคงความหรูหราและเข้ากับธีมมืดของระบบ เราจะใช้ไล่โทนสีกึ่งโปร่งแสงตามค่าระดับความถี่การเดินทาง:
>     - **จังหวัดที่ไม่มีการเดินทาง (0 ทริป)**: แสดงด้วยสีเทาอมน้ำเงินจางๆ `rgba(30, 41, 59, 0.15)`
>     - **จังหวัดที่มีประวัติการเดินทาง**: ไล่สีด้วย HSL Gradient จากสีฟ้าคราม/น้ำเงินอมม่วง (Indigo `#6366f1` / `hsla(240, 85%, 60%)`) ไปยังสีม่วงชมพูสดใส (Pink/Magenta `#ec4899` / `hsla(320, 85%, 50%)`) ตามอัตราส่วนทริปสูงสุดในแต่ละปีงบประมาณ
>     - **เมื่อผู้ใช้ชี้เมาส์ (Hover)**: ไฮไลต์ขอบสีขาวหนาขึ้น และแสดงกล่องข้อมูลลอยตัว (Custom Tooltip) แสดงชื่อจังหวัดภาษาไทยและสถิติจำนวนทริปเดินทางอย่างสวยงาม

  - **แถบซ้าย (7 คอลัมน์ - `lg:col-span-7`)**: วางแผนที่ประเทศไทยในกรอบแก้วขนาดใหญ่ มีความสูง 620px

    2. **แผนภูมิวงกลมแสดงสัดส่วนผู้จองรถสูงสุด 5 อันดับแรก**: แสดงตัวแผนภูมิและตารางคำอธิบายประกอบ
- **สคริปต์แผนที่และการพิกัด**:
  - สร้างอ็อบเจกต์ตัวจับคู่ชื่อจังหวัด (Thai Name to English name in GeoJSON) ครบทั้ง 77 จังหวัดแบบ 1:1
  - ดาวน์โหลดและอ่านข้อมูลจาก `/data/thailand_apisit.json` นำมาวาดลงบนแผนที่
  - ใช้ `map.fitBounds()` เพื่อปรับขนาดและโฟกัสแผนที่ให้อยู่ตรงกลางของประเทศไทยโดยอัตโนมัติ ไม่ว่าผู้ใช้จะเปิดผ่านหน้าจอใดๆ
  - พัฒนากล่องคำอธิบายข้อมูลลอยตามตัวชี้เมาส์ (Floating Tooltip) ที่จัดตำแหน่งได้ยืดหยุ่นในกรอบคอนเทนเนอร์

## Verification Plan

### Manual Verification
1. เปิดหน้า **แผนที่ความร้อน (Heat maps)** ในบราวเซอร์
2. ตรวจสอบว่ามีแผนที่ประเทศไทยแสดงผลอย่างสมบูรณ์แบบบนแถบซ้ายมือ และไล่เฉดสีจังหวัดตามจำนวนการเดินทางในตารางของปีงบประมาณนั้นๆ
3. ทดลองเลื่อนเมาส์ไปชี้ที่จังหวัดต่างๆ (เช่น กรุงเทพมหานคร, เชียงใหม่) เพื่อดูไฮไลต์ขอบสีและตรวจสอบว่าทูลทิปภาษาไทยแสดงจำนวนทริปตรงกันกับตารางอย่างแม่นยำ
4. ทดสอบเปลี่ยนตัวเลือกปีงบประมาณ แผนที่และเฉดสีไฮไลต์ต้องปรับเปลี่ยนตามข้อมูลปีที่เลือกทันที
5. ตรวจสอบการแสดงผลบนหน้าจอขนาดเล็ก (Responsive Layout) ว่าตัวแผนที่และตารางต่างๆ ยุบจัดวางในแนวตั้งได้อย่างเป็นระเบียบสวยงาม
